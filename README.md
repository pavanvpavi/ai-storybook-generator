# AI-Powered Personalized Children's Storybook Generator

An AI-powered web application that creates personalized children's storybooks with dynamic narratives and AI-generated illustrations.

## Features

- AI-generated personalized stories
- AI-generated illustrations
- Multiple story themes
- Flipbook-style reader
- Story saving with local storage
- Responsive design
- TypeScript + Next.js

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Groq API
- Hugging Face Inference API

## Setup

```bash
npm install
npm run dev
